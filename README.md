# Garg Medical Agencies - Veterinary Medical Store Website

## 🏥 Project Overview

A professional, modern, and minimal veterinary medical store website for **Garg Medical Agencies**, a trusted veterinary pharmacy located in Mathura, Uttar Pradesh, India. The website is designed to build brand presence, establish trust, and make it easy for farmers, veterinarians, and pet owners to contact and order from the store.
